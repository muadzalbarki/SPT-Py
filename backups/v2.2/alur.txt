PROJECT: SPT-DPRD (Sistem Otomatisasi Surat Pemerintahan)
=========================================================
Aplikasi desktop berbasis PySide6 untuk Sekretariat DPRD Kota Salatiga. 
Bertujuan mengotomatiskan pembuatan surat tugas (SPT), surat permohonan, 
dan surat kunjungan kerja menggunakan template .docx dengan placeholder 
{ } yang diisi secara dinamis. Database menggunakan SQLite + SQLAlchemy 
dengan fitur CRUD pegawai, manajemen template, generate surat, ekspor 
PDF (via LibreOffice), serta import/export Excel.

=========================================================
ALUR PENGGUNAAN APLIKASI
=========================================================
1. Kelola data pegawai di menu PEGAWAI (tambah/edit/import/export)
2. Upload template .docx di menu TEMPLATE (placeholder auto-detected)
3. Generate surat di menu GENERATE (pilih template, isi form, pilih peserta)
4. Lihat hasil surat di menu RIWAYAT SURAT
5. Ekspor surat ke PDF jika diperlukan

=========================================================
PENJELASAN SETAP MENU
=========================================================

1. DASHBOARD (Beranda)
   - Halaman utama dengan 4 kartu statistik:
     Total Pegawai, Template Tersedia, Riwayat Surat, Surat Hari Ini
   - Quick Actions: tombol shortcut ke Generate, Tambah Pegawai, 
     Upload Template, Export PDF
   - Tabel 10 surat terbaru (No Surat, Tanggal, Template, Peserta)

2. PEGAWAI (Data Pegawai)
   - CRUD data pegawai DPRD: Nama, NIP, Jabatan, Pangkat/Gol, 
     Komisi, No. HP
   - Pencarian real-time (nama/NIP/jabatan)
   - Import data dari Excel (.xlsx/.xls)
   - Export data ke Excel
   - Edit data via double-click pada tabel
   - Dialog modal untuk tambah/edit pegawai

3. TEMPLATE (Manajemen Template)
   - Upload template dokumen .docx
   - Deteksi otomatis semua placeholder { } dalam template
   - Daftar template tersimpan (kiri) dan daftar placeholder (kanan)
   - Menampilkan jumlah placeholder per template

4. GENERATE (Generate Surat)
   - Inti dari aplikasi - pembuatan surat otomatis
   - Langkah-langkah:
     a. Pilih template dari dropdown
     b. Pilih Komisi (A/B/C)
     c. Input nomor surat (auto-generated: 094/{kode}/{bulan_romawi}/{tahun})
     d. Pilih tanggal surat (date picker)
     e. Isi detail tujuan kunjungan: 
        Kab/Kota, Provinsi, Hari, Tanggal Kunjungan, Pukul, 
        Materi, Tentang, Dasar, Untuk, Rincian Jumlah
     f. Pilih peserta via Participant Checklist (collapsible per komisi)
     g. Klik "Generate Surat" - sistem mengganti semua placeholder 
        dengan data dari form + data pegawai terpilih
   - Progress bar selama proses generate
   - Hasil: file .docx tersimpan + record di database

5. RIWAYAT SURAT (Riwayat Surat)
   - Daftar semua surat yang telah digenerate (max 100 record)
   - Kolom: No Surat, Tanggal, Template, Peserta, File (status)
   - Tombol Refresh untuk memperbarui daftar
   - Tombol Export PDF (memerlukan LibreOffice terinstal)
   - Sorting descending berdasarkan tanggal pembuatan

6. PENGATURAN (Settings)
   - Tampilan: Tombol "Ganti Tema" (mode gelap/terang)
     Dark: Catppuccin Mocha, Light: Catppuccin Latte
   - Tentang Aplikasi: versi (v1.0.0), deskripsi, organisasi
     (Sekretariat DPRD Kota Salatiga)
